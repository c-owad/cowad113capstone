I made this work using Expo Snack (a website in which you can paste React Native code) on a linux laptop and an Android phone. 
I believe it should work on any Android phone as long as Expo Go is installed and it can scan the QR code generated by Expo Snack.
The sole dependancy is AsyncStorage for react which keeps track of the best scores on each of the game modes (RPG vs. standard).
It is a typing test game that allows you to either play an standard mode (type a sequence of random words or texts, each pulled from separate APIs) or
play RPG mode, which allows you to play against enemies that are trying to out-type you and you lose health incrementally as they out-type you or 
take health away from enemies as you out-type them (both of these are per correct letter typed). RPG mode has easy/normal/hard which is a % of your best WPM on the mode,
and both RPG and standard mode have the option to require you to type perfectly to win, or allow for typos.
I'm most proud of the overall UI/UX, as it feels very intuitive and fluid. Aside from that, I think using Async Storage was also very nice because it
keeps track of your best WPMs for each mode, which also allows for the RPG difficulties to be based off your skill.
